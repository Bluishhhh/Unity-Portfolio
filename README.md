# 🎮 Dark Fantasy RPG (Co-op | 3D Isometric)

> ⚠️ **Проект находится в активной разработке.**  
> Механики, интерфейс и структура могут меняться.  
> Следите за обновлениями в этом репозитории.

**Жанр:** Co-op RPG / Dark Fantasy / Survival  
**Движок:** Unity 6.3 LTS (URP)  
**Платформа:** PC  
**Перспектива:** 3D Isometric (фиксированный угол)  
**Игроки:** до 5 человек (кооператив)  
**Статус:** Активная разработка

---

## 🎯 Ключевые особенности

### ⚔️ Боевая система (реалтайм)
- Комбо-атаки (3 удара с множителями)
- Блок и парирование (с окном парирования)
- Hitbox-контроллер с расчётом:
  - Критического урона (от удачи)
  - Отбрасывания (Knockback)
  - Оглушения врагов (Stun)

### 📊 Система характеристик
- 7 атрибутов: Сила, Восприятие, Выносливость, Харизма, Интеллект, Ловкость, Удача
- Реальное влияние на:
  - HP / Stamina / Mana
  - Dodge Chance / Crit Chance
  - Carry Weight (влияет на инвентарь)
- Окно прокачки с превью изменений (зелёные цифры)

### 🎒 Инвентарь и экипировка
- Слоты: оружие (2 руки), броня (6 слотов), аксессуары (3 слота)
- Система веса (лимит зависит от силы)
- Стаки предметов (ресурсы, расходники)
- Тултипы с полной информацией о предмете

### 👾 ИИ врагов
- Преследование игрока (NavMesh / физика)
- Атаки с задержкой и хитбоксом
- Реакция на урон: оглушение + отбрасывание

### 💀 Система опыта и прогрессии
- Опыт за убийство врагов (визуальный сбор души)
- Повышение уровня (4 очка характеристик)
- Автоматическое восстановление Stamina 

### 🎮 Кооперативная основа (до 5 игроков)
- Архитектура закладывается под мультиплеер
- Каждый игрок — отдельный StatsModule
- UI адаптируется под текущего игрока

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `WASD` | Движение (изометрическое) |
| `ЛКМ` | **Атака** (зависит от оружия) |
| `ПКМ` | **Блок / Парирование** (если оружие позволяет) |
| `Space` | Рывок (с расходом стамины) |
| `Q` | **Умение 1** (зависит от оружия) |
| `E` | **Умение 2** (зависит от оружия) |
| `R` | **Врождённая способность** (классовая / расовая) |
| `F` | **Взаимодействие** (подбор, диалоги, активация) |
| `M` | **Карта** (локация / миникарта) |
| `I` | Инвентарь |
| `P` | Прокачка характеристик |

## ⚔️ Система оружия и умений (прогрессия)

Каждое оружие имеет **свой набор умений**, зависящий от:
- **Типа оружия** (меч, посох, лук, топор, кинжал и т.д.)
- **Редкости** (определяет количество и силу умений)
- **Модификаторов** (легендарные свойства, усиления)

### 📊 Редкость → количество умений

| Редкость | Умения | Описание |
|----------|--------|----------|
| **Common** (серый) | Только `ЛКМ` | Базовая атака. Блок отсутствует. Оружие для старта. |
| **Uncommon** (зелёный) | `ЛКМ` + `ПКМ` | Появляется блок / парирование. Оружие уже можно использовать в защите. |
| **Rare** (синий) | `ЛКМ` + `ПКМ` + `Q` | Добавляется активное умение (например, "Круговой удар", "Огненная стрела"). |
| **Epic** (фиолетовый) | `ЛКМ` + `ПКМ` + `Q` + `E` | Два активных умения. Оружие становится полноценным инструментом боя. |
| **Legendary** (оранжевый) | `ЛКМ` + `ПКМ` + `Q` + `E` + **система прокачки** | Оружие можно прокачивать отдельной веткой умений (как в Minecraft / Diablo). |

---

### 🗡️ Примеры умений по типу оружия

| Тип оружия | Пример умений |
|------------|---------------|
| **Меч** | `Q` — Круговой удар (АОЕ) / `E` — Рывок с ударом |
| **Посох** | `Q` — Огненный шар / `E` — Щит маны |
| **Лук** | `Q` — Заряженный выстрел / `E` — Дождь стрел |
| **Топор** | `Q` — Сокрушающий удар (броня) / `E` — Кровавая ярость |
| **Кинжал** | `Q` — Скрытый удар / `E` — Уклонение + контратака |

---

### 🏆 Легендарные предметы (отдельная система)

- Легендарные предметы имеют **уникальные модификаторы**, которые меняют поведение умений:
  - *«Меч Тысячи Ударов»* — `Q` наносит урон 3 раза подряд
  - *«Посох Бесконечности»* — `E` не расходует здоровье, если <50%
- Легендарное оружие можно **прокачивать**:
  - Улучшение базовых параметров (урон, скорость)
  - Открытие новых модификаторов
  - Ветка прокачки как отдельное дерево (сырая концепция)
